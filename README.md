# ☕ Ethiopian Coffee Game (Bunna Quest)

A fun memory matching game inspired by Ethiopian coffee culture 🇪🇹☕  
Built with Python and Kivy, this game challenges players to match coffee-related cards across multiple levels.

---

## 🎮 Features

- 🧠 Memory card matching gameplay
- 🇪🇹 Ethiopian coffee-themed symbols (BUNA, JEBENA, SIDAMA, etc.)
- 📈 Progressive difficulty levels (grid expands per level)
- 🔊 Sound effects (correct, wrong, shuffle)
- 🎨 Dark coffee-themed UI design
- 📱 Android-ready (Buildozer compatible)

---

## 🕹️ How to Play

1. Tap a card to reveal it
2. Find matching coffee pairs
3. Match all pairs to complete the level
4. Advance through 10 increasing difficulty levels
5. Finish the full coffee journey ☕

---

## 🧱 Tech Stack

- Python 🐍
- Kivy UI Framework 🎨
- Buildozer (for Android packaging 📱)
